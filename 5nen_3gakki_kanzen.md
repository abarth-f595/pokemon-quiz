## ■ 単元2：伝わる表現を選ぼう（敬語）

### 学習内容

**敬語の3種類**

| 種類 | 意味 | 使い方 |
|------|------|--------|
| 尊敬語（そんけいご） | 相手の行動・状態を高めて言う | 先生・目上の人の行動に使う |
| 謙譲語（けんじょうご） | 自分の行動をへりくだって言う | 自分・自分側の人の行動に使う |
| 丁寧語（ていねいご） | ていねいに言う | 「です」「ます」「ございます」 |

**尊敬語・謙譲語の対応表（重要）**
| 普通の言い方 | 尊敬語 | 謙譲語 |
|------------|--------|--------|
| 言う | おっしゃる | 申す・申し上げる |
| 行く | いらっしゃる・おいでになる | 参る・伺う |
| 来る | いらっしゃる・おいでになる | 参る・伺う |
| いる | いらっしゃる | おる |
| 食べる・飲む | 召し上がる | いただく |
| もらう | — | いただく・頂戴する |
| あげる・やる | — | 差し上げる |
| くれる | くださる | — |
| する | なさる | いたす |
| 見る | ご覧になる | 拝見する |
| 知っている | ご存じ | 存じる・存じ上げる |
| 聞く | — | 伺う・拝聴する |
| 読む | — | 拝読する |
| 思う | — | 存じる |

**丁寧語の使い方**
- 文末に「〜です」「〜ます」「〜ございます」をつける
- 名詞の前に「お〜」「ご〜」をつける（お名前・ご住所・お電話）
- ただし外来語には「お」「ご」をつけない（×おパソコン）

**場面別の敬語の使い方**
| 場面 | 普通の言い方 | 敬語 |
|------|------------|------|
| 先生が教室に来た | 先生が来た | 先生がいらっしゃいました |
| 自分が先生の家に行く | 先生の家に行く | 先生のお宅に伺います |
| 先生がお茶を飲む | 先生がお茶を飲む | 先生がお茶を召し上がる |
| 自分がお茶をもらった | もらった | いただきました |
| 先生に話を聞く | 聞く | 伺います・お聞きします |

### 応用問題

## ■ 単元3：方言と共通語

### 学習内容

**共通語と方言**
- 共通語：全国どこでも通じる言葉（かつては「標準語」と呼ばれた）
- 方言：ある地域だけで使われている言葉・言い方

**方言の特徴**
1. アクセント（イントネーション）が違う
2. 語彙（言葉そのもの）が違う
3. 文法・語尾が違う

**全国の方言の例**
| 地域 | 方言 | 共通語 |
|------|------|--------|
| 北海道 | なまら | とても |
| 北海道 | いたましい | もったいない |
| 北海道 | したっけ | そうしたら・じゃあね |
| 東北（青森） | どさ | どこへ |
| 東北（秋田） | あめゆ | あられ・みぞれ |
| 東京 | じゃん | じゃないか |
| 関西（大阪） | ちゃう | 違う |
| 関西（大阪） | おおきに | ありがとう |
| 関西（京都） | どす | です |
| 九州（博多） | ばってん | しかし・でも |
| 九州（熊本） | よか | いい |
| 沖縄 | はいさい | こんにちは（男性） |
| 沖縄 | めんそーれ | いらっしゃい |

**方言の価値と役割**
- 地域のアイデンティティ・文化を伝える
- 地域の歴史・生活の知恵が込められている
- 観光資源にもなる
- コミュニティのつながりを深める

**共通語と方言の使い分け**
- 改まった場（授業・発表・放送）→共通語
- 地域・家族・友人との会話→方言でもよい
- 目的や相手に応じて使い分けることが大切

| 種類 | 特徴 | 例 |

## ■ 単元6：複合語

### 学習内容

**複合語とは**
- 2つ以上の言葉が組み合わさってできた言葉
- それぞれの意味が合わさって新しい意味を作る

**複合語の種類**

**① 和語＋和語**
- 例：雨雲（あめ＋くも）・春風（はる＋かぜ）・朝日（あさ＋ひ）
- 例：草花（くさ＋はな）・夕暮れ（ゆう＋ぐれ）・手紙（て＋かみ）

**② 漢語＋漢語**
- 例：図書館（としょ＋かん）・運動場（うんどう＋じょう）
- 例：科学者（かがく＋しゃ）・音楽家（おんがく＋か）

**③ 和語＋漢語・漢語＋和語**
- 例：朝食（あさ＋しょく）・夕食（ゆう＋しょく）
- 例：本棚（ほん＋たな）・砂浜（すな＋はま）

**④ 外来語を含む複合語**
- 例：スーパーマーケット（スーパー＋マーケット）
- 例：バスケットボール（バスケット＋ボール）
- 例：アイスクリーム（アイス＋クリーム）

**複合語の読み方変化（連濁）**
- 2つ目の言葉の最初の音が濁音に変わることがある
- 例：花（はな）＋火（ひ）→花火（はなび）※「ひ」→「び」
- 例：秋（あき）＋空（そら）→秋空（あきぞら）※変化なし

## ■ 単元1：単位量あたりの大きさ

### 学習内容

**単位量あたりの意味**
- 「1つあたり」「1㎡あたり」「1人あたり」の量を求めて比べる
- 単位を揃えないと比較できないときに使う

**人口密度**
```
人口密度（人/km²）＝ 人口 ÷ 面積（km²）
```
- 例：A市 人口6000人・面積200km²→ 6000÷200＝30人/km²
- 例：B市 人口8000人・面積400km²→ 8000÷400＝20人/km²
- → A市の方が混んでいる

**混み具合の比べ方**
- 1㎡あたりの人数 ＝ 人数 ÷ 面積
- 例：A教室 30人・80㎡ → 30÷80＝0.375人/㎡
- 例：B教室 24人・60㎡ → 24÷60＝0.4人/㎡ → B教室の方が混んでいる

**1あたりの量のまとめ**
| 求めたいもの | 公式 |
|------------|------|
| 人口密度 | 人口 ÷ 面積 |
| 1㎡あたりの人数 | 人数 ÷ 面積 |
| 1人あたりの広さ | 面積 ÷ 人数 |
| 1mあたりの値段 | 値段 ÷ 長さ |
| 1kgあたりの値段 | 値段 ÷ 重さ |
| 1Lあたりの走行距離 | 距離 ÷ 燃料 |

**2量の関係を逆から求める**
- 単位量 × 個数（量）＝ 全体量
- 全体量 ÷ 単位量 ＝ 個数（量）

## ■ 単元2：割合

### 学習内容

**割合の意味と公式**
```
割合 ＝ 比べる量 ÷ もとにする量

比べる量 ＝ もとにする量 × 割合

もとにする量 ＝ 比べる量 ÷ 割合
```

**「くもわ」の公式（覚え方）**
```
    く
  も  わ
く（比べる量）＝ も（もとにする量）× わ（割合）
```

**百分率（パーセント）**
- 割合を100倍にして%で表す
- 割合0.01 ＝ 1%、割合0.1 ＝ 10%、割合1 ＝ 100%
- 変換：割合 → % ：×100 → %
- 変換：% → 割合 ：÷100 → 割合

**歩合（ぶあい）**
- 割合を10倍した単位
| 歩合 | 割合 | 百分率 |
|------|------|--------|
| 1割 | 0.1 | 10% |
| 2割 | 0.2 | 20% |
| 5割 | 0.5 | 50% |
| 1割5分 | 0.15 | 15% |
| 7割3分 | 0.73 | 73% |
| 3割2分5厘 | 0.325 | 32.5% |

**割合の3用法（例題）**

**①割合を求める**
- 定員40人のうち30人が合格 → 割合 = 30÷40 = 0.75 = 75%

**②比べる量を求める**
- 定価2000円の品物を30%引きで売る → 売値 = 2000×0.7 = 1400円

**③もとにする量を求める**
- ある学校の5年生の40%が80人 → 全体 = 80÷0.4 = 200人

**割引・割増の計算**
- 20%引き → ×0.8（100%-20%=80%=0.8）
- 15%引き → ×0.85
- 30%増し → ×1.3（100%+30%=130%=1.3）

## ■ 単元3：帯グラフと円グラフ

### 学習内容

**帯グラフ**
- 長方形（帯）を各項目の割合に応じて区切ったグラフ
- 全体を100%として各部分の割合を示す
- 複数の帯を並べると割合の変化を比べやすい

**帯グラフの読み方**
1. 最も割合が大きい項目を読む
2. 各項目の%を読む
3. 2つ以上の帯グラフを比べるときは、同じ項目を見比べる

**円グラフ**
- 円を各項目の割合に応じて区切ったグラフ
- 全体を100%（360°）として表す
- 各項目の角度 ＝ 割合 × 360°

**円グラフの作り方**
1. 各項目の割合（%）を求める
2. 割合 × 360 ＝ その項目の角度（°）
3. 大きい順（または決められた順）に区切る
4. 「その他」は最後

**帯グラフ・円グラフの使い分け**
| グラフ | 適した場面 |
|--------|-----------|
| 帯グラフ | 複数の時期・集団の割合を比べるとき |
| 円グラフ | 1つの全体の内訳を示すとき |

**計算例**
全体500人のうち、A：200人、B：150人、C：100人、その他：50人
- A：200÷500＝0.4＝40%　→ 角度：40×360÷100＝144°
- B：150÷500＝0.3＝30%　→ 角度：108°
- C：100÷500＝0.2＝20%　→ 角度：72°
- その他：50÷500＝0.1＝10%　→ 角度：36°

## ■ 単元4：正多角形と円

**正多角形の定義と性質**
- 正多角形：すべての辺の長さが等しく、すべての角の大きさが等しい多角形

| 名前 | 辺の数 | 1つの角 | 中心角 |

| 正十角形 | 10 | 144° | 36° |
| 正n角形 | n | (n-2)×180÷n | 360÷n |

**正多角形のかき方（円を使う）**
1. 円をかく
2. 円の中心角を辺の数でわる（正六角形なら360÷6＝60°）

```
円周 ＝ 直径 × 円周率（π ≒ 3.14）
```

- π（パイ）≒ 3.14159265…
- 小学校では3.14を使う

**円の面積の応用**
- 半円の面積 ＝ 半径 × 半径 × 3.14 ÷ 2
- 1/4円の面積 ＝ 半径 × 半径 × 3.14 ÷ 4
- ドーナツ型（リング）＝ 大きい円 − 小さい円

## ■ 単元5：角柱と円柱

### 学習内容

**角柱の定義と種類**
- 角柱：2つの合同で平行な多角形（底面）と、いくつかの長方形（側面）で囲まれた立体

| 名前 | 底面の形 | 頂点の数 | 辺の数 | 面の数 |
|------|---------|---------|--------|--------|

| 五角柱 | 五角形 | 10 | 15 | 7 |
| 六角柱 | 六角形 | 12 | 18 | 8 |
| n角柱 | n角形 | 2n | 3n | n+2 |

**円柱の定義**
- 円柱：2つの合同で平行な円（底面）と、1つの曲面（側面）で囲まれた立体
- 頂点：なし　底面：2つの円　側面：1つの長方形（展開したとき）

**展開図**
- 三角柱の展開図：三角形2つ＋長方形3つ

**円柱の展開図の計算**
- 側面の長方形の横の長さ ＝ 底面の円周 ＝ 2 × 半径 × 3.14
- 側面の長方形の縦の長さ ＝ 円柱の高さ

**角柱・円柱の見分け方**
- 底面が多角形 → 角柱
- 底面が円 → 円柱
- 上下が同じ形・平行 → 柱（さらに先がとがっていたら錐・すい）

### 学習内容

**速さの定義と公式**
```
速さ ＝ 距離 ÷ 時間
距離 ＝ 速さ × 時間
時間 ＝ 距離 ÷ 速さ
```

**「はじき」の公式（覚え方）**
```
  は（距離）
じ    き
じ（時間）　き（速さ）
距離 ＝ 速さ × 時間
速さ ＝ 距離 ÷ 時間
時間 ＝ 距離 ÷ 速さ
```

**速さの単位**
| 単位 | 読み方 | 意味 |
|------|--------|------|
| m/秒 | メートル毎秒 | 1秒に進むメートル数 |
| m/分 | メートル毎分 | 1分に進むメートル数 |
| km/時 | キロメートル毎時 | 1時間に進むキロメートル数 |

**単位換算**
```
時間：1時間 ＝ 60分 ＝ 3600秒
距離：1km ＝ 1000m

km/時 → m/分 ：×1000÷60　または　÷0.06
m/分 → km/時 ：×60÷1000　または　×0.06
m/秒 → km/時 ：×3600÷1000 ＝ ×3.6
km/時 → m/秒 ：÷3.6
```

**時速・分速・秒速の換算例**
- 時速72km → 分速：72000÷60＝1200m/分 → 秒速：1200÷60＝20m/秒
- 秒速5m → 分速：5×60＝300m/分 → 時速：300×60÷1000＝18km/時

**追い越し・出会い問題**
- 向かい合う（出会い）：2人の速さの和 × 時間 ＝ 距離
- 同じ方向（追い越し）：速さの差 × 時間 ＝ 距離

**溶解（ようかい）の言葉**
| 語句 | 意味 | 例 |
|------|------|-----|
| 溶質（ようしつ） | 溶けるもの | 食塩・砂糖・ミョウバン |
| 溶媒（ようばい） | 溶かすもの | 水 |
| 溶液（ようえき） | 溶質が溶媒に溶けたもの | 食塩水・砂糖水 |

**水溶液の特徴**
1. 透明（色がついていても透明なものは溶液）
2. どこも同じ濃さ（均一）
3. 時間が経っても沈殿しない

**溶解度（ようかいど）**
- 溶解度：水100gに溶けることができる溶質の最大量（g）
- 溶解度は水の温度によって変わる

**食塩とミョウバンの溶解度の違い**
| 水の温度 | 食塩（NaCl） | ミョウバン |
|---------|------------|---------|
| 20℃ | 約36g | 約12g |
| 40℃ | 約37g | 約24g |
| 60℃ | 約37g | 約57g |
| 80℃ | 約38g | 約321g |

→ 食塩は温度が上がっても溶解度がほとんど変わらない
→ ミョウバンは温度が上がると溶解度が大きく増える

**飽和水溶液（ほうわすいようえき）**
- それ以上溶けることができない水溶液

**溶けたものを取り出す方法**
| 方法 | 原理 | 適した溶質 |
|------|------|----------|
| 蒸発させる | 水を蒸発させ溶質を残す | 食塩（温度に関係なく取り出せる）|
| 冷やす（再結晶） | 温度を下げて溶解度を下げる | ミョウバン（温度で大きく変化するもの）|
| ろ過（ろか） | ろ紙で固体を取り除く | 溶けきれなかった固体の分離 |

**ろ過の方法**
1. ろうとにろ紙をセットする（ろ紙は三角形に折る）
2. ガラス棒を使って液をろうとに流す
3. ろ液（ろ紙を通った液）を集める

**水溶液の重さ**
- 食塩が水に溶けても、全体の重さは変わらない
- 水100g + 食塩10g = 食塩水110g
- 溶けた食塩は見えなくなるが、質量は保存される（質量保存の法則）

## ■ 単元2：ふりこのきまり

### 学習内容

**ふりこの各部の名前**
- ふりこ：ひもの先におもりをつけてぶら下げたもの
- 支点（してん）：ひもを固定している点
- ふれはば：支点の真下から左右にふれた角度（または距離）
- 1往復：左端→右端→左端（または右→左→右）の動き

**ふりこの周期（1往復する時間）に影響する条件**

**実験①：ふりこの長さを変える**
| ふりこの長さ | 1往復する時間 |
|-----------|------------|
| 短い | 短い（速い） |
| 長い | 長い（遅い） |
→ ふりこの長さが長いほど、1往復する時間は長くなる

**実験②：おもりの重さを変える**
| おもりの重さ | 1往復する時間 |
|-----------|------------|
| 軽い | 変わらない |
| 重い | 変わらない |
→ おもりの重さは周期に影響しない

**実験③：ふれはばを変える**
| ふれはば | 1往復する時間 |
|---------|------------|
| 小さい | 変わらない |
| 大きい | 変わらない |
→ ふれはばは周期に影響しない

**まとめ：ふりこの周期を変えるのは「ふりこの長さ」だけ**

**ふりこの法則の応用**
- 振り子時計：ふりこの長さを調節して時間を正確に刻む
- ふりこが長い→1往復に時間がかかる→時計が遅くなる
- ふりこが短い→1往復が速い→時計が速くなる

**対照実験の重要性**
- 調べたい条件だけを変えて、他の条件はすべて同じにする
- 例：ふりこの長さを調べる→重さとふれはばは同じにする

## ■ 単元3：電磁石の性質

### 学習内容

**電磁石とは**
- 鉄芯（コア）にコイルを巻き、電流を流すと磁石の性質が生まれる
- 電流を流しているときだけ磁石になる（電流を切ると磁力がなくなる）

**電磁石の極**
- 電磁石にはN極とS極がある
- 電流の向きを変えると、N極とS極が入れ替わる
- 永久磁石と同じく「同極はしりぞけ合い、異極は引き合う」

**電磁石の強さを変える方法（2つ）**
| 方法 | 結果 |
|------|------|
| コイルの巻き数を増やす | 電磁石が強くなる |
| 電流を強くする（電池を増やす） | 電磁石が強くなる |

**コイルの巻き数と磁力の関係**
- 巻き数が多い → 磁力が強くなる（釘が多くつく）
- 巻き数が少ない → 磁力が弱くなる

**電流の大きさと磁力の関係**
- 電流が大きい（電池を増やす）→ 磁力が強くなる
- 電流が小さい → 磁力が弱くなる

**電磁石と永久磁石の比較**
| 比較項目 | 電磁石 | 永久磁石 |
|---------|--------|---------|
| 磁力 | 電流を流しているときのみ | 常にある |
| 極の切り替え | 電流の向きを変えれば可能 | 不可能 |
| 強さの調節 | 可能（巻き数・電流） | 不可能 |
| 引きつけるもの | 鉄・ニッケルなど | 鉄・ニッケルなど |

**電磁石の利用例**
| 利用例 | 仕組み |
|--------|--------|
| モーター | 電流の向きが変わり、コイルが回転する |
| スピーカー | 電流の変化で振動板が振動する |
| 電磁ブレーキ | 鉄を引きつけて止める |
| MRI（医療機器） | 強力な電磁石を使う |
| リニアモーターカー | 電磁石の反発・引力で浮上・推進 |
| クレーン（鉄くず工場） | 電磁石で鉄くずを持ち上げ、電流を切って離す |

  - 部品の多くは関連工場が作っている

**トヨタ生産方式（ジャスト・イン・タイム）**
- 必要なものを・必要なときに・必要なだけ供給する
- 無駄なストックを減らし、効率的に生産する
- 「かんばん方式」ともいわれる

**自動車の輸出・貿易**
- 主な輸出相手国：アメリカ・ヨーロッパ・オーストラリア
- 輸送手段：専用の自動車運搬船（カーキャリア）
- 現地生産：貿易摩擦を避けるため、現地の工場で生産することも増えた

**環境対応車の開発**

| プラグインハイブリッド（PHV） | 外部電源で充電できるHV |
