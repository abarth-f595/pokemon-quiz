## ■ 単元1：かんがえるのって おもしろい（詩）

### 学習内容

**詩の特徴と種類**
- 詩は「行分け」で書かれ、普通の文章とは形が違う
- 詩の種類：口語詩（現代の言葉）／文語詩（昔の言葉）
- 詩の形式：定型詩（音数が決まっている）／自由詩（音数が自由）

**詩の表現技法**
| 技法名 | 意味 | 例 |
|--------|------|-----|
| 比喩（直喩） | 「〜のような」「〜みたいな」で例える | 雲が綿のようにふわふわだ |
| 比喩（隠喩） | 「〜だ」と断言して例える | 君は太陽だ |
| 擬人法 | 物や動物を人のように表す | 風が笑う |
| 反復法 | 同じ言葉を繰り返す | 青い、青い、空 |
| 体言止め | 名詞で行を終わらせる | 静かな春の朝 |
| 倒置法 | 言葉の順序を逆にする | 美しい、この花は |

**詩を読むときのポイント**
1. 声に出して音読する
2. 情景を思い浮かべる
3. 作者が何を伝えたいか考える
4. 表現の工夫を見つける

## ■ 単元2：漢字の成り立ち

### 学習内容

**漢字の成り立ち4種類**

#### ① 象形文字（しょうけいもじ）
- 物の形をそのまま絵にして作った文字
- 例：日（太陽の形）、月（三日月の形）、山（山の形）、川（川の流れ）、木（木の形）、目（目の形）、口（口の形）、耳（耳の形）、手（手の形）、火（炎の形）、水（水の流れ）、鳥（鳥の形）、魚（魚の形）

#### ② 指事文字（しじもじ）
- 点や線で場所・数などの概念を表した文字
- 例：上（線の上に点）、下（線の下に点）、一（横一本）、二（横二本）、三（横三本）、本（木の根元に点）、末（木の先端に点）、中（真ん中を示す）

#### ③ 会意文字（かいいもじ）
- 2つ以上の文字を組み合わせて新しい意味を作った文字
- 例：
  - 林（木+木 → たくさんの木）
  - 森（木+木+木 → もっとたくさんの木）
  - 明（日+月 → 明るい）
  - 休（人+木 → 人が木に寄りかかって休む）
  - 岩（山+石 → 山の大きな石）
  - 男（田+力 → 田んぼで力仕事をする人）
  - 好（女+子 → 母と子 → 好き）
  - 字（宀+子 → 家の中で子が学ぶ）

#### ④ 形声文字（けいせいもじ）
- 意味を表す部分（意符）と読み方を表す部分（音符）を組み合わせた文字
- 例：
  - 語（言+吾）→ 言が意味、吾（ゴ）が読み
  - 晴（日+青）→ 日が意味、青（セイ）が読み
  - 海（水+毎）→ 水が意味、毎（マイ）が読み
  - 軽（車+圣）→ 車が意味、圣が読み
  - 銅（金+同）→ 金が意味、同（ドウ）が読み
  - 清（水+青）→ 水が意味、青（セイ）が読み
  - 打（手+丁）→ 手が意味、丁が読み

**部首（ぶしゅ）について**
| 部首名 | 形 | 意味 | 例 |
|--------|-----|------|-----|
| にんべん | 亻 | 人に関すること | 休・使・作・仮 |
| さんずい | 氵 | 水に関すること | 海・池・泳・清 |
| きへん | 木 | 木に関すること | 根・松・植・橋 |
| くさかんむり | 艹 | 植物に関すること | 花・草・茶・葉 |
| てへん | 扌 | 手の動作に関すること | 打・投・持・押 |
| ごんべん | 言 | 言葉に関すること | 語・話・読・記 |
| こころ | 心 | 心・気持ちに関すること | 思・想・急・忙 |
| やまへん | 山 | 山に関すること | 峰・岸・岳・崎 |

## ■ 単元4：きいて、きいて、きいてみよう

### 学習内容

**インタビューの基本**
インタビューとは：相手に質問して、その人のことや考えを引き出す会話

**インタビューをする人（聞き手）のポイント**
1. 事前に質問を準備する（5〜8つ程度）
2. 相手の話をしっかりうなずきながら聞く
3. 聞き取れなかったら「もう一度教えてください」と丁寧に確認する
4. メモは要点だけ書く（全部書こうとしない）
5. 話が広がった場合は追加質問をする（「もう少し教えてもらえますか」）

**インタビューをされる人（話し手）のポイント**
1. 相手の目を見て話す
2. 具体的なエピソードを交えて話す
3. 短く分かりやすく伝える

**記録する人のポイント**
1. 要点を整理しながらメモする
2. 「いつ・どこで・誰が・何を・どうした」を押さえる
3. 話し手の言葉を正確に書き留める

**質問の組み立て方**
- オープン質問（はい/いいえで終わらない）：「どう思いますか」「どんな経験がありますか」
- クローズド質問（はい/いいえで答える）：「好きですか」「行ったことはありますか」
- フォローアップ質問：話の続きを引き出す「それはなぜですか」「もう少し詳しく教えてください」

## ■ 単元5：見立てる（説明文）

### 学習内容

**「見立て」とは**
- 「見立て」とは：ある物を別の物に見なしたり、例えたりすること
- 日本文化に古くからある感覚（枯山水、生け花、盆栽など）
- 例：石を山に見立てる、砂を海に見立てる

**説明文の構造（三段構成）**
| 構成 | 内容 |
|------|------|
| 序論（はじめ） | 話題の提示・問いかけ |
| 本論（なか） | 具体例・根拠・詳しい説明 |
| 結論（まとめ） | まとめ・筆者の主張 |

**要旨のつかみ方**
1. 序論と結論をしっかり読む（筆者の主張は結論にある）
2. 「つまり」「したがって」「このように」などの接続語の後に注目
3. 繰り返し出てくる言葉がキーワード

**接続語の種類**
| 接続語 | 働き | 例 |
|--------|------|-----|
| だから・したがって | 理由→結論 | 〜だから、〜だ |
| しかし・でも・ところが | 逆接 | 〜しかし、〜だ |
| また・そして | 付け加え | 〜また、〜も |
| つまり・すなわち | まとめ | つまり、〜ということだ |
| たとえば | 例示 | たとえば、〜がある |

### 応用問題

**問題1**　次の文章の「要旨」（筆者が最も伝えたいこと）を50字以内でまとめなさい。

```
人はものを「見立てる」という行為を自然に行っている。
たとえば、雲を動物に見立てたり、木の形を人の姿に見立てたりする。
これは日本の伝統文化にも見られ、生け花では花を自然の風景に見立てる。
このように見立ては、日本人が古くから大切にしてきた豊かな想像力の表れである。
```

## ■ 単元6：言葉の意味が分かること（説明文）

### 学習内容

**内容のポイント**
- 筆者：今井むつみ（言語学者）
- テーマ：言葉は単に「名前」ではなく、概念とセットで学ばれる
- 言葉を知ることは、その言葉が指す「もの・こと」の概念を理解すること
- 例：「りんご」という言葉を知っても、りんごという概念（赤い・丸い・甘い・果物）を理解していないと本当の意味で分かっていない

**「言葉の概念」の例**
- 「犬」という言葉：4本足・毛がある・吠える・ペットになる…など多くの概念を含む
- 「左・右」の概念：子どもが「左」と「右」を区別するのは難しい（方向という概念が必要）
- 言葉は文化・経験とつながって意味が深まる

**説明文の読み取り**
- 問い（疑問文）を見つける → 答えを探す
- 具体例がある部分と、筆者の考えの部分を区別する

### 応用問題

**問題1**　「言葉の意味が分かる」とはどういうことですか。この説明文の内容をもとに60字以内でまとめなさい。

## ■ 単元8：日常を十七音で（俳句創作）

### 学習内容

**俳句作りの手順**
1. テーマ（場面・感動した瞬間）を決める
2. そのテーマの季語を調べる
3. 伝えたい「景色・音・感触・気持ち」を書き出す
4. 五・七・五に当てはめる
5. 言葉を工夫して整える

**俳句の推敲ポイント**
- 「ている」「しました」は省ける場合が多い
- 季語は1つだけ（2つ入ると「季語の重複」になる）
- 体言止めや切れ字を効果的に使う

**切れ字の使い方**
| 切れ字 | 効果 | 例 |
|--------|------|-----|
| や | 感動・強調（上の句の終わり） | 古池や〜 |
| かな | 余情・詠嘆（下の句の終わり） | 〜水の音かな |

## ■ 単元4：小数のかけ算

### 学習内容

**小数 × 整数**
方法：小数を整数に直してかけ、最後に小数点を戻す
例：2.3 × 4
→ 23 × 4 = 92
→ もとの小数点の位置に戻す：9.2

**小数 × 小数**
方法：両方の小数を整数に直してかけ、最後に小数点を戻す
例：2.3 × 1.4
→ 23 × 14 = 322
→ 小数点以下の桁数の合計（1+1=2桁）分、左に点を移す：3.22

**小数点の移動ルール**
- かけ算では、2つの数の小数点以下の桁数の「合計」分、積の小数点を左に移す

**計算例**
```
3.6 × 2.5
= 36 × 25 ÷ 100
= 900 ÷ 100
= 9.0 = 9
```

**積が1より小さくなる場合**
- 1より小さい数をかけると、積はもとの数より小さくなる
- 例：5 × 0.6 = 3.0（5より小さい）
- 1より大きい数をかけると、積はもとの数より大きくなる

**面積への応用**
例：縦1.5m、横2.4mの長方形の面積
= 1.5 × 2.4 = 3.60 = 3.6m²

- 肥料（肥料がないと成長が悪い）
